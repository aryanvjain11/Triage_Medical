Triage AI

An AI-driven medical prioritization assistant designed to streamline emergency room and clinical workflows by assessing patient urgency.

Project Overview
Triage AI analyzes patient symptoms, vital signs, and intake data to categorize cases by medical urgency. The goal of this project is to reduce waiting times for critical cases and assist healthcare professionals in high-volume environments.

Features
Urgency Classification: Automatically categorizes patient intake data into priority levels (e.g., Critical, Urgent, Non-Urgent).
Symptom Parsing: Uses natural language processing to extract key health indicators from unstructured patient descriptions.
Risk Score Estimation: Provides a confidence metric to help triage nurses validate the AI's assessment.
